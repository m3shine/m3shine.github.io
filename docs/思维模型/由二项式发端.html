二项式

(a+b)n (a+b)^n (a+b)n

(a+b)0 = 1
(a+b)1 = a+b
(a+b)2 = (a+b)(a+b) = (a+b)a+(a+b)b = a2+2ab+b2
(a+b)3 = (a+b)2(a+b) = (a2+2ab+b2)(a+b) = a3+3a2b+3ab2+b3

二项式系数三角

从(a+b)0开始，一直到(a+b)n，所有二项式的系数可以形成一个杨辉三角（帕斯卡三角）。

杨辉（南宋）1261年在《详解九章算法》中引用贾宪（北宋）的《释锁算书》，后书片段录入《永乐大典》，存于英国剑桥大学图书馆或大英博物馆。
帕斯卡，1654年发现该三角。

有了这个三角，系数可以不用再计算组合，从逻辑计算思维转到直觉思维，根据幂对应的行直接得出各项系数。

(a+b)n二项式的各项系数对应的就是杨辉三角的第n行（n从0开始！）

杨辉三角的特点

  • 每行左右两端都是1，同行数字大小左右对称
  • 从第2行起，除左右两端，每个数是它上面两数之和
  • 第n行有n+1项
  • 还有很多很多特点需要去发现…

func YangHuiTriangle(lines int) {
    var nums []int
    for i := 0; i < lines; i++ {
        fmt.Print("<div class='row'>")
        for j := 0; j < (i + 1); j++ {
            length := len(nums)
            var value int
            if j == 0 || j == i {
                value = 1
            } else {
                value = nums[length-i] + nums[length-i-1]
            }
            nums = append(nums, value)
            if value%2 == 0 {
                fmt.Print("<span class='box_red'>", value, "</span>")
            } else {
                fmt.Print("<span class='box_blue'>", value, "</span>")
            }
        }
        fmt.Println("</div>")
    }
}

牛顿广义二项式

(P+PQ)m/n (P+PQ)^{m/n} (P+PQ)m/n

注意，牛顿二项式是把二项式的第二项转化为包含第一项的形式。
而m/n可以是整数也可以是分数，可以是正数也可以是负数，即任意实数幂，此所谓广义。

例： 1(1+x)53\frac{1}{\sqrt[3]{(1+x)^5}}3(1+x)5​1​ = (1+x)-5/3，这个特例里P是1

牛顿二项式展开式

(P+PQ)m/n=Pm/n+mnAQ+m−n2nBQ+m−2n3nCQ+m−3n4nDQ+⋯ (P+PQ)^{m/n} = P^{m/n}+\frac{m}{n}AQ+\frac{m-n}{2n}BQ+\frac{m-2n}{3n}CQ+\frac{m-3n}{4n}DQ+\cdots (P+PQ)m/n=Pm/n+nm​AQ+2nm−n​BQ+3nm−2n​CQ+4nm−3n​DQ+⋯

A、B、C、D…代表前一项

二项分布

二项式与概率的联系
二项分布是独立重复事件，离散分布

具备以下条件的事件就满足二项分布

  • n次重复相互独立
  • 每次概率p固定
  • 结果只有两种

二项分布描述的是n次事件中，有k次出现期望结果的概率

比如，连续抛10次硬币，正面(或反面)朝上的次数可能是[0,10]里的任何一个数，这个数记为k，叫随机变量，发生k次的概率服从二项分布。

P(nk)=n!k!(n−k)!pk(1−p)n−k P{n\choose k} = \frac{n!}{k!(n-k)!}p^k(1-p)^{n-k} P(kn​)=k!(n−k)!n!​pk(1−p)n−k

这就是二项式定理里的一项，即第k项，其中，

n!k!(n−k)!\frac{n!}{k!(n-k)!}k!(n−k)!n!​ 是k次结果的数量，

pk(1−p)n−kp^k(1-p)^{n-k}pk(1−p)n−k 是每个结果的概率。

第k项的系数（结果数量）乘以单次概率等于n次里发生k次结果的概率。

举例

还是掷硬币，掷10次，出现5次正面（或反面）朝上的概率是多少？

(105){10\choose 5}(510​)(0.5)5(1-0.5)10-5 = 10!5!\frac{10!}{5!}5!10!​(0.5)10 = 0.24609375

意思是掷10次硬币，有24.6%的概率可以得到5次正面（或反面）朝上的结果。

不是应该50%吗？反直觉了吗？
我们接着算，4次正面（或反面）朝上的概率：

(104){10\choose 4}(410​)(0.5)4(1-0.5)10-4 = 10!4!\frac{10!}{4!}4!10!​(0.5)10 = 0.205078125

掷10次硬币，有20.5%的概率可以得到4次正面（或反面）朝上的结果。

极限

什么是极限？
从牛顿的逐渐消失的量到莱布尼茨的无限小，到伯克莱主教拷问的消失的量的鬼魂，到拉格朗日用级数推导导数避免"无限",再到柯西割圆术的无限趋近并用不等式证明无限的存在，再到魏尔斯特拉斯没有时间空间及几何的直观的代数不等式断言，历经几个世纪才有了严格的极限定义：

∀ϵ>0,∃N∈N,∀n>N,∣xn−α∣<ϵ \forall \epsilon>0,\exists N \in \mathbb {N},\forall n>N,|x_n-\alpha|< \epsilon ∀ϵ>0,∃N∈N,∀n>N,∣xn​−α∣<ϵ

对任意ε>0，总存在自然数N，使得当n>N时，不等式 |xn-α\alphaα|<ε 恒成立。
这就是说当n无限增大时，xn无限接近于α\alphaα，记为 lim⁡n→∞xn=α\displaystyle\lim _{n\rightarrow\infty}{x_n}=\alphan→∞lim​xn​=α
