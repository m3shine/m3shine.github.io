---

- 结构  
结构化思维的核心特征就是——联系。清楚理解事物由几部分组成，各部分之间是如何联系的。本质是逻辑，将零散的数据、信息、知识、思维等有机组织起来。

- 方法  
方法是研究的手段，归纳演绎、抽象具体等辩证  
自学的关键是用自己的话讲(写)出来

- 模型  
模型是对规律的总结

- 框架  
框架是思维的脚手架，是具体反复可用的

- 架构  
架构是平衡利弊的决策
